# CallPilot — Project Summary

*The Negotiator · ElevenLabs × Hack-Nation. ~200 words for the jury.*

---

Every phone-priced market runs on the same unfairness: the same 45-mile move is quoted anywhere from **$1,158 to $6,506** — a **5.6× spread** for identical work — and the only defense is calling five to eight companies, describing your job identically each time, and haggling. Almost nobody has the hours, and sight-unseen phone quotes end up 40% higher than promised (FMCSA).

**CallPilot is a voice agent that does the calling for you.** It runs the same three beats every time:

**Estimate** — a voice interview (ElevenLabs Agents) and document upload (OpenAI vision) build one structured job spec you confirm; it is then read verbatim on every call, so the quotes are genuinely comparable.

**Call** — the agent negotiates three distinct counterparties (a tough negotiator, a stonewaller who won't quote over the phone, and a fee-hiding lowballer), extracting an itemized quote from each. The conversations are real and unscripted: the price moves because of what the agent says, not because a script told it to.

**Close** — it calls the others back citing the real cheapest quote as leverage (never a fabricated number), applies red-flag rules (30%-below-market, hidden deposits), and returns a ranked report with transcript evidence and a plain-language recommendation.

It discloses that it is an AI, never invents inventory or bids, and ends every call in a structured outcome. Retargeting from movers to auto repair or medical bills is one config file — the whole system is config-driven.
